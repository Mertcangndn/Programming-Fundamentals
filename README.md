# 🚀 Görev 2: BST'yi Çift Yönlü Bağlı Listeye Dönüştürme

Bu proje, verilen bir İkili Arama Ağacı (BST) yapısını, hafızada yeni yer ayırmadan (in-place), inorder sıralamasına göre Çift Yönlü Bağlı Listeye (DLL) dönüştürür.

Bu proje, üniversite veri yapıları dersi ödevi ("Ağacı Düzleştir") amacıyla oluşturulmuştur.

---

## 🛠️ Kullanılan Teknoloji

* **C**

---

## ⚙️ Kurulum (Getting Started)

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyebilirsiniz.

### Gereksinimler

* GCC derleyicisi (veya herhangi bir C derleyicisi)

### Adımlar

1.  Bu repoyu klonlayın:
    ```sh
    git clone [https://github.com/](https://github.com/)[kullanici-adiniz]/[repo-adiniz].git
    ```
2.  Proje dizinine gidin:
    ```sh
    cd [repo-adiniz]
    ```
3.  Kodu derleyin (Örn: `main.c` dosyanızın adıysa):
    ```sh
    gcc main.c -o bst_to_dll
    ```

---

## 🏃‍♀️ Kullanım

Programı çalıştırmak için:

```sh
./bst_to_dll
