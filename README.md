# Programming Fundamentals 🚀

A consolidated repository containing my early programming projects, algorithm implementations, and data structure exercises. 

This workspace reflects my foundational journey in computer engineering, showcasing my progression from basic console applications to complex data structure manipulations. It serves as a unified archive of my foundational coursework and personal practice projects.

## 🗂️ Repository Structure

The projects in this monorepo are primarily written in **C** and **Python** and can be categorized into three main areas:

### 1. Data Structures & Memory Management (C)
Core exercises focusing on pointers, dynamic memory allocation, and custom data structures.
* `BST-to-List`: Converting a Binary Search Tree to a doubly linked list.
* `ListNode-Workspace`: Implementations and operations on singly linked lists.
* `NodePolidromStack`: Checking if a sequence is a palindrome using stack structures and nodes.

### 2. Algorithmic Checks & Logic (C & Python)
Functions and scripts designed to solve specific logical or mathematical problems.
* `Armstrong-Number-Checker-Function-for-C`: Efficiently checks for Armstrong numbers.
* `Smith-Number-Checker-Function-for-C`: Identifies Smith numbers based on prime factorization.
* `Is-the-point-inside-the-circle-Python-TurtleLib`: A visual geometric calculator using Python's Turtle library.

### 3. Mini Applications & OOP Practice (C & Python)
CLI-based applications focusing on Object-Oriented Programming principles, user input validation, and basic system design.
* `simple-bank-system`: A foundational banking simulator with deposit/withdrawal logic.
* `StudentGradeManager` & `CS-student-average-calculator`: Tools for managing student records and applying OOP class structures.
* `Phone-Dictionary`: A terminal-based contact management system.
* `Calculator`: A fundamental calculator implementation.
